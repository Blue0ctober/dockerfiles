# docker-utorrent

## Description

A Dockerfile for [uTorrent](http://www.utorrent.com/).

## Volumes

### `/data`

Configuration files for uTorrent.

### `/downloads`

Download directory for torrents.

## Ports

### 53160

Torrent TCP port.

### 53160/udp

Torrent UDP port.

### 8080

WebUI port.
